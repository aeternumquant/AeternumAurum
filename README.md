# Aeternum Aurum Partners

**Plataforma Privada de Inteligência de Capital**

[![GitHub Repo](https://img.shields.io/badge/GitHub-Repository-black?style=for-the-badge&logo=github)](https://github.com/aeternumquant/AeternumAurum)

### Sobre o Projeto

A **Aeternum Aurum Partners** é uma sociedade privada de gestão de capital com disciplina institucional.  
Não somos um fundo tradicional. Nosso mandato duplo é:

- **Preservação de Patrimônio** contra riscos sistêmicos (desvalorização monetária e volatilidade de commodities)
- **Expansão de Capital** através de estratégias quantitativas proprietárias

**Pilares estratégicos:**
- Global Macro
- Event-Driven
- Quantitative Risk

### O que você encontra neste repositório

- Site oficial completo (Next.js + TypeScript)
- Design premium dark mode com estética institucional
- Toda a apresentação institucional (slides e conteúdo)
- Framework proprietário **AAIF** (Aeternum Auction Intelligence Framework)

### Tecnologias Utilizadas

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (animações suaves)
- **Canvas / WebGL** para visualizações avançadas de liquidez e gamma
